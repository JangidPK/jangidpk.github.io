This project explores molecular structure generation using deep generative models. Molecular structures are represented as SMILES strings and modeled using Recurrent Neural Networks (RNNs) and Transformer architectures to generate novel molecules through probabilistic sampling.

## Features

* SMILES-based molecular representation
* RNN and Transformer models
* Deep learning with PyTorch
* Probabilistic molecule generation
* Evaluation of generated molecular structures

## Tools

* Python
* PyTorch
* NumPy
* pandas
* RDKit

## Applications

* Drug discovery
* Molecular design
* Computational chemistry
* Generative AI for scientific research

## Repository

This project demonstrates the application of modern sequence modeling techniques to molecular generation and chemical space exploration.
